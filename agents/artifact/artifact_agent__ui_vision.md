\# Artifact Agent — UI Vision



\## Visual Identity

Artifact should feel:

\- powerful

\- clean

\- dark

\- fast

\- premium

\- less like a popup and more like a serious workspace



\## Layout Feel

Artifact should not feel cramped.



It should support:

\- large primary view area

\- optional secondary panes

\- clear but quiet controls

\- room for charting and analysis without feeling cluttered



\## Toolbox Feel

The toolbox should be:

\- small

\- discreet

\- easy to reach

\- not visually dominant

\- capable enough to feel like a Swiss Army knife



The user wants tools such as:

\- text

\- draw

\- straight trendline

\- highlight

\- crop

\- zoom

\- share/export

\- reset



These should be available without making Artifact look like drawing software.



\## Chart Feel

Artifact charts should feel consistent even when using different libraries.



Shared design language:

\- dark background

\- subtle grid treatment

\- consistent typography

\- consistent spacing

\- coherent hover and selection styling

\- consistent control chrome



\## Interaction Feel

Artifact should feel:

\- responsive

\- stateful

\- persistent

\- smooth

\- safe to explore



The user should be able to:

\- open it

\- inspect something

\- annotate if desired

\- save mental context

\- return later and still feel continuity
