\# Artifact Agent — Surface Vision



\## Surface Purpose

Artifact is where the user goes when a symbol, structure, or idea deserves serious study.



It should answer:

\- What is this actually doing?

\- What does this look like visually?

\- How does this trade behave?

\- What happens if I change IV, time, or structure?

\- How does this compare to alternatives?

\- What should I preserve for later?



\## Surface Structure

Artifact should feel like a flexible analysis operating surface.



It may contain:

\- a main chart/view area

\- optional side rail for artifact switching

\- optional detachable/resizable panes

\- discreet toolbox

\- session context / metadata area if needed



\## Surface Modes

Artifact should be able to support multiple modes:

\- price-first mode

\- risk-first mode

\- comparison mode

\- annotation/review mode

\- memory/recall mode



\## Relationship to Other Surfaces

\- Watchlist feeds Artifact

\- Scanner feeds Artifact

\- Positions may feed Artifact

\- Journal may receive outputs from Artifact later



Artifact is not a replacement for those surfaces.

It is the escalation destination.



\## User Goal

The user should feel:

“This is where I go when I want to actually think.”
