\# Scanner Agent — Outputs



\## Primary Outputs

The Scanner Agent should produce:



\### 1. Scanner Panel Surface

A Scanner panel with:

\- top mode toggle: Entry / Roll

\- compact filter strip

\- dense results table

\- lower-third mini vol surface



\### 2. Entry Scanner Results

Results for new capital deployment that compare structures by normalized risk / actual capital effect.



\### 3. Roll Scanner Results

Results for rolling that evaluate:

\- current structure

\- what is left behind

\- same-underlying interactions

\- new candidate structure

\- resulting holistic impact



\### 4. Mini Vol Surface

A compact 3D or pseudo-3D volatility surface for the selected underlying showing the next 5–10 expirations.



\### 5. Scanner Data Structures

Typed structures for:

\- scanner mode

\- candidate rows

\- roll candidate rows

\- selected underlying

\- mini surface payload

\- ranking metadata



\## Secondary Outputs

Optional outputs may include:

\- row detail drawer

\- candidate explanation tooltip

\- compact richness view

\- scanner/vol-surface click linkage



\## Non-Outputs

The Scanner Agent should not output:

\- Positions interaction code

\- Watchlist hierarchy code

\- Artifact workspace code

\- Journal system code

\- Alerts delivery code



\## Delivery Format

The Scanner Agent must return:

\- full file replacements only

\- explicit file paths

\- no partial snippets unless explicitly requested
