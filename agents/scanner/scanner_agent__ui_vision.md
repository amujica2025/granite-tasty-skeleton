\# Scanner Agent — UI Vision



\## Visual Identity

Scanner should feel:

\- dense

\- fast

\- professional

\- table-first

\- analytical

\- not decorative



\## Top Region Feel

The top region should feel like a serious control strip:

\- Entry / Roll mode switch is obvious

\- filters are compact

\- mode context changes are clear



\## Results Table Feel

The middle table should feel like:

\- a sortable opportunity board

\- directly comparable rows

\- right-aligned numeric values

\- compact enough to scan quickly

\- not card-based



\## Lower Vol Surface Feel

The lower-third mini vol surface should:

\- read quickly in small space

\- look like live pricing context, not decoration

\- update immediately when selected underlying changes

\- support quick interpretation of nearby term structure



\## Interaction Feel

\- selecting a row should feel immediate

\- changing mode should clearly change the panel’s purpose

\- sorting should be obvious

\- Scanner should feel like a control surface, not a report



\## User Experience Goal

The user should feel:

“I can compare entries or rolls quickly, see how the chain is priced, and decide without guesswork.”
