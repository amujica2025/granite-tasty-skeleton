\# Watchlist Agent — UI Vision



\## Visual Identity

The Watchlist should feel like part of a professional trading workstation:

\- dark

\- compact

\- table-first

\- data-dense

\- not decorative

\- not consumer-dashboard styled



\## Parent Row Feel

Collapsed watchlist parent rows should feel:

\- heavier than child rows

\- structurally important

\- quickly scannable

\- visually distinct from ticker rows



They should resemble market buckets or expandable list groups.



\## Child Row Feel

Expanded ticker rows should feel:

\- tight

\- live

\- comparable

\- cleanly aligned for numeric reading



These are scan rows, not cards.



\## Numeric Alignment

\- symbols left aligned

\- numeric values right aligned

\- compact headers

\- consistent width behavior where practical



\## Chart Feel

The lower-third mini chart should:

\- update immediately on highlight

\- visually match the Artifact chart language

\- be minimal and fast

\- not try to replace Artifact



\## Interaction Feel

\- click should feel instant

\- expand/collapse should be immediate

\- row highlight should be obvious but not loud

\- double click should clearly escalate to Artifact



\## User Experience Goal

The user should feel:

“I can move through groups quickly, focus on names fast, preview price action immediately, and decide whether to escalate to Artifact.”
